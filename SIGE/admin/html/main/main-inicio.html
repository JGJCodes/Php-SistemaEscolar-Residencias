Bienvenido al MODO ADMINISTRADOR.

Administración de usuarios.

El Sistema Integral de Gestión Escolar cuenta con un modulo de gestión de usuarios para administrar las cuentas de las personas que acceden al sistema.

Reportes

El Sistema Integral de Gestión Escolar cuenta con la opción de exportar sus reportes administrativos en tres diferentes formatos: docx, pdf, xls.

Diseño responsivo

El Sistema Integral de Gestión Escolar cuenta con una interfaz de usuario adaptable para distintos dispositivos y navegadores web.
